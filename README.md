# Warehouse Robot Simulator

This project is a full-stack simulation of warehouse robots that pick up and deliver items on a grid.

## Features
- Simulates robots moving on a warehouse floor (grid)
- Task assignment and prioritization
- Real-time updates in the browser
- Pathfinding algorithm to reach tasks

## Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express
- Optional: WebSockets for real-time updates

## How to Run
1. Clone the repo
2. Run backend: `cd backend && npm install && npm start`
3. Run frontend: `cd frontend && npm install && npm start`
4. Open browser at `http://localhost:3000`

## Skills Demonstrated
- Full-stack development
- Algorithms (pathfinding)
- Real-time systems
- Frontend visualization
